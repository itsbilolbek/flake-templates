# Generic Nix Flake

A modern development template using flake-parts, treefmt, and pre-commit hooks.

## Get Started

1. Enter the environment:

```bash
nix develop
```

2. Format everything:

```bash
nix fmt
```

This automatically installs git hooks and loads your tools.

3. Verify the flake:

```bash
nix flake check
```
